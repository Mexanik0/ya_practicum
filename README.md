# Проекты Яндекс.Практикум

За время обучения c июля 2023 по февраль 2024 года подготовлено 14 проектов.

**Инструменты**:
`Jupyter Notebook` `Python` `Pandas` `Matplotlib` `SciPy` `Seaborn` `Tableau` `MS PowerPoint`

**Основные навыки**:
`обработка данных` `исследовательский анализ данных` `статистический анализ данных` `проверка гипотез` `A/B-тест` `визуализация данных` `составление презентаций` `построение дашбордов`
